##### PREREQUISITES
* Python 3.x 
* pip3
* git

##### PYTHON 3 PREREQUISITES
* aiohttp
* asyncio

## Installation
```sh
$ git clone https://github.com/penucuriCode/kcfinder
$ cd kc-find
$ pip3 install -r requirements.txt
$ python exploit.py
```
##### Update Wordlist
```sh
$ apt install wget
$ python update.py
```
#### Note
Error 403 = Firewall in Website <br />
Error 500 = Server not Responding / Connection Error
